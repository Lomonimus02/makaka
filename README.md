# Makaka - Курсы по Вайбкодингу

Лендинг для продажи курсов по "Вайбкодингу" с историей успеха.

## Технологии
- HTML5
- CSS3 (Custom Properties, Flexbox, Grid)
- Vanilla JavaScript

## Запуск
Просто откройте файл `index.html` в любом современном браузере.

## Структура
- `index.html` - Разметка страницы
- `style.css` - Стили (темная тема, неоновые акценты)
- `script.js` - Интерактивность (плавная прокрутка, обработка кнопок)
